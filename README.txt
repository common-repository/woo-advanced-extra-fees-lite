== Installation ==

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won't need to leave your web browser. To do an automatic install of WooCommerce Advanced Extra Fees Lite, log in to your WordPress dashboard, navigate to the Plugins menu, and click Add New.
 
In the search field type "WooCommerce Advanced Extra Fees Lite." then click Search Plugins.Once you've found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click Install Now, and WordPress will take it from there.

This section describes how to install the plugin and get it working.

= Manual installation =

e.g.

1. Upload `woocommerce-advanced-extra-fees.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Installation Instructions =
* Download the plugin
* Upload the folder woocommerce-advanced-extra-fees-lite to wp-content/plugins (or upload a zip through the WordPress admin)
* Activate and enjoy!

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.

If you encounter issues with the shop/category pages after an update, flush the permalinks by going to WordPress > Settings > Permalinks and hitting Save. That should return things to normal.
